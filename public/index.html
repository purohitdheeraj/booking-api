Welcome to Our Booking API

This API allows you to manage event organizers, customers, events, tickets, and bookings.

Endpoints

  • POST /organizers/register - Register an event organizer
  • POST /organizers/login - Log in as an event organizer
  • GET /events - List events
  • POST /events - Create an event (organizers only)
  • PUT /events/:id - Update an event (organizers only)
  • DELETE /events/:id - Delete an event (organizers only)
  • POST /bookings - Book a ticket (customers only)

For detailed information, please refer to our API documentation.
